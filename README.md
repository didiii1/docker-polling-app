# 📊 Docker Polling App

Aplikasi voting sederhana berbasis Flask yang terhubung dengan PostgreSQL menggunakan Docker.

---

## 🔧 Teknologi
- Flask
- PostgreSQL
- Docker
- Docker Compose

---

## ▶️ Cara Menjalankan

```bash
docker-compose up -d

Buka browser:
http://localhost:5000

✨ Fitur
Voting pertanyaan
Melihat hasil voting
Penyimpanan data menggunakan database (persistent)

🧠 Kesimpulan

Project ini membantu memahami penggunaan Docker untuk menjalankan aplikasi berbasis Flask dengan database PostgreSQL. Selain itu, dipelajari juga konsep persistensi data menggunakan Docker volume.

Kesulitan yang dialami adalah konfigurasi container dan database, namun dapat diatasi dengan memahami log dan dokumentasi Docker.
